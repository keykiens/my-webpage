Контрольные вопросы — Лабораторная работа №8 (DOM)

1. Какие методы DOM используются для динамического создания элементов списка задач?
Используются методы:
  • document.createElement() — создание новых элементов;
  • appendChild() — добавление элементов в DOM;
  • classList.add() — добавление CSS‑классов;
  • textContent — установка текста внутри элемента.
2. Как добавляется обработчик события для кнопок?
Через метод addEventListener(), например:
button.addEventListener("click", function() { ... });
3. Как проверить чтобы поле ввода не было пустым перед добавлением новой задачи?
Используется проверка с trim():
if (taskInput.value.trim() === "") return;
4. Каким образом выполненные задачи визуально отличаются от невыполненных?
Выполненным задачам добавляется класс completed, который в CSS задаёт:
  • зачёркнутый текст (text-decoration: line-through;)
  • серый цвет (color: #888;)
5. Как можно расширить функциональность этого приложения для сохранения задач между сеансами?
Использовать localStorage:
  • localStorage.setItem() — сохранить список задач;
  • localStorage.getItem() — загрузить задачи при открытии страницы;
  • данные сохраняются даже после перезагрузки браузера.
