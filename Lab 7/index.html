Проверка функций

Контрольные вопросы

Числа

Вопрос: Как проверить, является ли число целым, и какие методы JavaScript можно использовать для этой проверки?

Ответ: Можно использовать Number.isInteger(num), который возвращает true, если число целое. Также можно проверить через num % 1 === 0.

Строки

Вопрос: Как извлечь подстроку из строки, начиная с определённой позиции и заданной длины?

Ответ: Для этого используют метод substr или substring. Пример:

function getSubstring(str, start, length) {
    return str.substr(start, length);
}
console.log(getSubstring("Hello World", 0, 5)); // "Hello"

Булевы значения

Вопрос: Что такое операция "исключающее ИЛИ" (XOR), и как она может быть реализована для двух булевых значений?

Ответ: XOR возвращает true, если значения разные. В JavaScript можно реализовать так:

function xor(a, b) {
    return (a || b) && !(a && b);
}
console.log(xor(true, false)); // true
console.log(xor(true, true));  // false

Объекты

Вопрос: Как проверить, содержит ли объект определённое свойство, и в чём разница между использованием in и hasOwnProperty?

Ответ: - Оператор in проверяет наличие свойства в объекте и его прототипах.
- Метод hasOwnProperty проверяет только собственные свойства объекта.

Массивы и их обработка

Вопрос: Чем отличаются методы forEach и map при работе с массивами?

Ответ: - forEach выполняет действие для каждого элемента массива, но не возвращает новый массив.
- map создаёт новый массив, преобразуя каждый элемент по заданной функции. 

const arr = [1, 2, 3];
arr.forEach(x => console.log(x * 2)); // выводит 2, 4, 6

const doubled = arr.map(x => x * 2);
console.log(doubled); // [2, 4, 6]
