Контрольные вопросы по CSS Grid

1. Какими свойствами задаются количество и ширина столбцов в Grid‑контейнере?

Используются свойства grid-template-columns и grid-template-rows. Например: grid-template-columns: repeat(3, 1fr);.

2. Какой метод можно использовать, чтобы равномерно распределить доступное пространство?

Для этого применяют функцию fr (fractional unit). Она делит пространство пропорционально. Например: grid-template-columns: 1fr 1fr 1fr;.

3. Какие функции CSS Grid позволяют автоматически подстраивать количество колонок или строк?

Функции auto-fill и auto-fit в сочетании с minmax(). Например: grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));.

4. Каким свойством можно задать промежутки между элементами Grid?

Используется свойство gap (или отдельно row-gap и column-gap).

5. Как разместить элемент в нескольких строках или столбцах Grid?

Применяются свойства grid-column и grid-row. Например: grid-column: 1 / 3; — элемент займёт два столбца.
